Algorithm Deep Dive

Dijkstra's Shortest Path

How it works

  • Initialize dist[source]=0, all others=∞
  • Extract minimum distance unvisited node
  • Relax all neighbors: if shorter path found, update
  • Mark node visited, repeat until destination reached
  • Reconstruct path using predecessor array

Use in our project

  • Emergency vehicle routing (fastest path)
  • Accident rerouting (avoid blocked roads)
  • Alternate route calculation
  • City road network: 8 nodes, 11 edges
